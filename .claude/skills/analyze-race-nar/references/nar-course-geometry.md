# NAR コース物理形状カタログ（静的・正本／lazy 確定）

> **役割**: JRA の `course-geometry.md` の NAR 版。観点 D・E・展開合成はここを参照し **web で再調査しない**（I10）。
> 「当該開催の今」（砂厚・含水率・凍結防止剤・内外の砂質差）は**変動情報**＝従来どおり D が当日 web で採る。
> **数値は△暫定**（初期値・要一次ソース確認）。初回にその場を分析するとき**該当場だけ** web で確認し、確定したら△を外して
> `確認済 YYYY-MM` を付す（lazy 確定＝全場を先回りで調べない）。帯広（ばんえい）は対象外。

## NAR 共通の展開原則（全場にかかる基本骨格）

- **小回り×短直線＝先行有利が基本骨格**。JRA より「テンのダッシュ力→好位確保」の価値が高く、直線一気は構造的に届きにくい。
  展開合成は前残り系を本線側に置くバイアスから出発し、崩れる条件（先行過多・ハイ消耗）を明示的に探す。
- **コーナー4回のツーターン戦が主流**＝道中の器用さ（コーナーでの加減速・内をロスなく回る技術）が A_cruise に効く。
- **砂質は場ごとに別物**（海砂/川砂/洗い砂・砂厚）。転入・遠征馬の「初砂質」は割引候補（観点N の転入初戦割引と重複計上しない＝D は砂質適合の実績だけ見る）。
- ナイター開催（大井・船橋・川崎・門別・高知等）は馬場の乾湿が昼と変わる＝当日情報。
