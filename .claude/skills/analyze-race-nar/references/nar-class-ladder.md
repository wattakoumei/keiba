# NAR クラス階梯カタログ（観点N「クラス格・転入換算」の静的正本）

> **役割**: 地方競馬（NAR）は**場ごとにクラス体系が違い**、混合戦・転入/移籍馬が常態＝「格の位置づけ」が最強の一次信号。
> JRA では `fetch_racecard.py` の `class_rank`/`class_move` が機械でやっていた仕事を、NAR ではこのカタログ＋観点N（`obs-n-class`）が担う。
> N はここの階梯・換算原則を spawn 注入で受け取り、web では**各馬の現級・転入元の確定だけ**を行う（原則の再調査はしない＝I10）。
> **as-of 2026-07・換算は暫定初期値**（一般に流布する目安ベース・NARコーパス較正前）。レビュー（/review-prediction）で換算外れが再発したら**この表を較正**する（lazy 蓄積）。

## 転入・移籍の換算原則（観点Nの読み筋）

1. **直近実走クラスで測る**＝全盛期でなく。長期休養明け転入・高齢の降級は直近内容まで割り引く。
2. **転入初戦は1段割引**（環境・砂質・小回り初経験）。特に JRA→地方の初戦は「格上でも取りこぼす」が定跡。
   **転入2〜3戦目が本領**＝初戦凡走→2戦目巻き返しのパターンを懸念/好材料の両面で書く。
3. **JRA→地方の換算は「JRAでの着順内容」で**: JRA で勝ち上がれなくても、未勝利で掲示板複数なら南関 C1〜C2 で即通用（上の表）。
   JRA 1勝クラス在籍歴は南関 B 級相当＝下級への転入なら大幅な格上。
4. **地方→地方は地区格差を掛ける**（上の暫定序列）。南関→他地区は格上げ方向、逆は格下げ方向で読む。
5. **昇降級の勢い**: 連勝中の昇級（賞金で強制昇級）は「格上げでも勢い優先」＝割引を小さく。
   負け続けての降級（クラス滑落）は「楽になる」より「下降基調」を先に疑う（obs-i の decline と突き合わせ）——**ただし原則7を先に確認**。
6. **セリ・募集価格は使わない**（それは新馬モード観点Mの領域）。ここは実走クラスの序列だけで測る。
7. **格上帰りの反転則（2026-07-10 川崎レビュー由来）**: 近走の崩れが**今回より上のランク（R差+1以上）で起きたもの**なら、
   下降基調・久々の減点だけで相手圏から外さない＝**格の証拠を直近着順より優先**する。NARはクラス間の能力差が急峻で、
   「格上で通用しなかった」は「現級で通用しない」の証拠にならない（B2で崩れてB3へ＝巻き返しが定跡）。
   圏外に消してよいのは (a)今回も格上挑戦 (b)現級以下でも崩れが続く (c)高齢衰え・故障明け等が複合する、のときだけ。
   原則1の「直近内容まで割り引く」は**格の水増しを防ぐ**ためであり、相手圏の資格まで奪わない（印側の床は analyze-race-nar SKILL の NAR 印規律）。

> **市場ゼロは不変**（I1）: 格の換算に**オッズ・人気・他人の予想を使わない**。使うのは公式格付け・実走クラス・着順/着差の事実のみ。
