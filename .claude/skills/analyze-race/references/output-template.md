## predictions.jsonl への追記フォーマット

レポート出力後、各買い目候補馬について1行ずつ追記。
`expected`（展開予想の要点）を入れておくと review 側で映像照合せず一次仕分けできる：

```json
{"race_id":"20260531-tokyo-11","race_name":"...","date":"2026-05-31","model_version":"1.1","horse_no":8,"horse":"△△","my_prob":0.14,"market_prob":0.07,"edge":0.07,"ev_win":1.68,"bet_types":["win","place"],"stake_pct":0.02,"expected":{"running_style":"差し","assumed_pace":"S","premise":"スロー前提・ハイ化で半減"}}
```

展開予想の全体（脚質分類表・隊列推移・反証条件）は `report.md` の §2 に保存され、
これが `/review-prediction` の A/B/C 仕分けの照合元になる。
