# スコアリングモデル（発揮能力 → 勝率 → 妙味 → 買い目）

> **このファイルが唯一の調整点。** 改善ループ（review-prediction）はここの重み・閾値を編集する。
> 変更時は末尾の「変更履歴」にバージョンと理由を追記すること。

## モデルバージョン: v1.1

各馬について、観点 A〜J の評価（-2..+2）を集約して発揮能力スコアを出す。

### 1. 観点評価の取り込み

各観点の馬ごと評価 `e_X ∈ [-2, +2]` を 0..1 に正規化：`n_X = (e_X + 2) / 4`。
（確信度「低」の観点は中央 0.5 方向へ 30% 収縮：`n_X ← 0.5 + (n_X - 0.5)*0.7`）

### 2. 潜在能力 base（0–100）

A・B・C の加重平均を 0–100 にスケール。

| 観点 | 重み |
|------|------|
| A 能力指数・時計 | 0.40 |
| B 近走内容・クラス通用度 | 0.40 |
| C 血統 | 0.20 |

```
base = 100 * (0.40*n_A + 0.40*n_B + 0.20*n_C)
```

### 3. 適性 multiplier（≈0.7–1.2）

D・E を中立 1.0 まわりに展開。`n` が 0.5 で 1.0、0 で下限、1 で上限。

| 観点 | 効き幅 |
|------|--------|
| D コース/距離/馬場適性 | ±0.15 |
| E 展開・脚質×枠 | ±0.10 |

```
apt = 1.0 + 0.15*(2*n_D - 1) + 0.10*(2*n_E - 1)   # ≈ [0.75, 1.25]
```

### 4. 状態 multiplier（≈0.77–1.23）

F・G・H・K を中立 1.0 まわりに展開。

| 観点 | 効き幅 |
|------|--------|
| F 調教・厩舎 | ±0.07 |
| G ローテ・馬体・間隔 | ±0.05 |
| H 当日気配・パドック | ±0.05 |
| K 騎手・乗り替わり | ±0.06 |

```
cond = 1.0 + 0.07*(2*n_F - 1) + 0.05*(2*n_G - 1) + 0.05*(2*n_H - 1) + 0.06*(2*n_K - 1)  # ≈ [0.77, 1.23]
```

> K は「乗り替わり方向 + 騎手のコース巧拙 + コンビ相性」を 0..1 に正規化した `n_K`。
> 前走騎手が取れず乗り替わり判定が保留の馬は確信度「低」→ §1 の収縮で自動的に中立寄りに。

### 5. 割引（減算）

I のスコア `e_I ∈ [-2, 0]` を base に対する減点に変換。

```
discount = base * 0.08 * (-e_I)    # e_I=-2 で base の16%減
```

### 6. 発揮能力スコア

```
ability = base * apt * cond - discount
```

ability が負になる場合は 0 でクリップ。

### 7. 自分の勝率（場全体で正規化）

softmax で勝率シェアに変換。温度 `T` で散らばりを調整（T 大 → 平坦）。

```
T = 14
p_i = exp(ability_i / T) / Σ_j exp(ability_j / T)
```

`p_i` が「自分の勝率」%。（複勝率・連対率の近似は §10 参照）

### 8. 市場勝率（overround 除去）

観点 J で集めた単勝オッズ `o_i` から：

```
raw_i = 1 / o_i
overround = Σ_j raw_j           # ≈ 1.25（単勝控除率20%相当）
m_i = raw_i / overround         # 市場勝率%
```

複勝の市場勝率は複勝オッズ下限を使うか、単勝市場勝率から §10 の近似で代用。

### 9. 妙味・期待値・買い目

```
妙味(edge) = p_i - m_i           # プラス=市場が過小評価
EV_単勝   = p_i * o_i            # 1.0 が損益分岐
```

**買い目判定（初期閾値）:**

- **単勝**: `EV_単勝 ≥ 1.10` かつ `edge ≥ +3pt` の馬を本命〜対抗候補。
- **複勝**: 複勝勝率（§10）× 複勝オッズで `EV_複勝 ≥ 1.05` の馬。手堅い妙味馬向け。
- **賭け金**: Kelly 比率の 1/4 を提示（`f = (p*o - 1)/(o - 1)` の 1/4）。上限は資金の5%。

### 10. 連対率・複勝率の近似（三連系拡張）

Harville 近似で勝率 `p_i` から着順分布を作る。

```
P(iが1着) = p_i
P(iが2着) = Σ_{k≠i} p_k * p_i/(1 - p_k)
P(iが3着) = Σ_{k≠i}Σ_{l≠i,k} p_k * p_l/(1-p_k) * p_i/(1-p_k-p_l)
複勝率 p3_i = P(1着)+P(2着)+P(3着)   （※18頭立て等の複勝3着圏）
```

**三連複**: 3頭組 {i,j,k} の確率 ≈ 全順列の Harville 和。
EV = 組確率 × 三連複オッズ。`EV ≥ 1.15` の組を点数を絞って推奨。

**三連単**: 順序付き (i→j→k) の Harville 確率。
EV = 順序確率 × 三連単オッズ。`EV ≥ 1.20`。点数が増えるので軸（1着固定/2着まで固定）を明示。

> 三連系は点数管理が肝。**合成オッズと総賭け金あたり期待回収**もレポートに出す。

### 11. 偶然（信頼区間）

点推定 `p_i` を過信しないため、ブレ幅を注記する。
観点の平均確信度が低い／頭数が多い／展開不確実が高いほど区間を広く。

```
σ_i ≈ p_i * (0.15 + 0.10*低確信度割合 + 0.05*(頭数>14 ? 1 : 0))
勝率レンジ = [p_i - σ_i, p_i + σ_i]（0未満は0でクリップ）
```
