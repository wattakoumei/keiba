# コース物理形状カタログ（静的・正本）

> **役割**: レース距離ごとの発走地点・初角までの距離・ゴール前直線・坂・ダートの芝スタート（内外枠差）など**不変のコース物理データの正本**。
> 観点 D・E・展開合成（STEP4a）はここを参照し、**web で再調査しない**（再現性のためのワンソース化）。
> 「当該開催の今」（馬場傾向・含水率・柵替わり）は変動情報なので従来どおり D が web で採る。
> **使い方**: STEP1 で場・コース・距離が確定したら**該当距離の行＋場の基本値**を抜いて spawn データ（レース条件）に同梱する。
> 2026-06-11 深掘り改訂（JRA公式・JRA-VAN・Wikipedia 等の複数ソース突合）。**△＝ソース間で差異あり/推定＝確信度を一段下げて使う**。直線長は A コース基準。

## ダート芝スタート早見（内外枠の芝距離差）

JRA で芝スタートのダートは次の **8コースだけ**。いずれも**外枠ほど芝部分を長く走れる**（ダッシュがつき先行しやすい）：

|---|---|---|
| 東京 ダ1600 | 内約150m／外約180m | 外が約30m長い＝外枠有利が明確 |
| 中山 ダ1200 | 約150m | 外枠有利。内枠は被されやすい |
| 京都 ダ1400 | 約150m△ | 外+約30m。枠の優劣は東京1600・阪神1400より小さい |
| 阪神 ダ1400 | 約150m△ | 外枠有利が顕著 |
| 阪神 ダ2000 | 約78m | 芝部分が短く影響は小 |

上記以外のダート（札幌・函館・小倉の全距離、各場の1800m系 等）は**すべてダート発走**＝芝スタートの枠差は無い。

## 参照ルール（DRY）

- **D（適性）**: 物理形状（直線・坂・初角・芝S）は本カタログが正本＝web 再調査しない。web で採るのは「当該開催の今」だけ。
- **E（展開証拠）**: 枠の有利不利の判定に「初角までの距離」と「芝スタート早見」を使う（初角が近い＝外枠不利／芝S＝外枠のダッシュ利）。
- **展開合成（STEP4a）**: `phase_flow` の因果文に該当コースの直線長・坂・初角を織り込む（例:「中山の急坂で甘い逃げは止まる」「東京芝2000は初角126mで外枠が位置を取れない」「函館は終盤下りで前が止まらない」）。
- 数値の更新（コース改修・訂正）はこのファイルだけ直す。
