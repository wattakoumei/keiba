# 血統カタログ（半静的・正本）

> **役割**: 主要種牡馬の**父としての産駒傾向**と**母父(BMS)としての供給傾向**の正本。観点 C（血統）はこれを参照し、**カタログにある血なら web 再調査しない**（再現性・トークン削減）。
> **半静的**: コース形状（不変）と違い種牡馬傾向は動く＝**年1回程度の更新**＋新種牡馬の産駒デビュー時に追記。`as-of 2026-06`。
> **使い方**: STEP1 で各馬の父・母父（seed にある）に該当する行を抜いて C に spawn 同梱。**カタログ外の血だけ C が web 調査**し、結果は追記候補。
> **決め手の質**列が相別マッピング直結: 瞬発→A_finish／持続・スタミナ→A_cruise+A_finish／先行力→A_early。
> 2026-06-12 新設（netkeiba/JBIS 産駒成績ベース・5並列調査＋実在検証）。**△＝産駒少で暫定／確信度低。市場は不使用**。

## 巻頭原則（血統判断の一次フィルタ・C と合成側が共有）

1. **すべて傾向＝一次フィルタ**。個体の近走・馬体・実績が血統を**上書き**する（血統は実績の無い条件の穴埋めが主用途＝[[observation-points]] C の定義）。
2. **「父スピード・母父スタミナ」は俗説**。実態は父母半々。**母父が新しいスピード種牡馬（ロードカナロア等）なら母父=スタミナは当てはまらない**。母父は系統名で決め打たず下表の「供給する質」を見る。
   （注: サクラバクシンオーは短距離種牡馬だが母系はステイヤー＝キタサンブラックの長距離G1はむしろ母系スタミナが出た例。系統名と母系の中身は別物。）
3. **黄金配合（ニックス）**は相性確率を上げる材料であって保証でない: ディープ×ストームキャット／×Unbridled's Song／×フレンチデピュティ、ステイゴールド×メジロマックイーン、キンカメ×サンデー。
4. **ダートのサイン**=父か母父に米国型（Mr.Prospector/Storm Cat/Unbridled's Song/Fappiano/French Deputy/A.P.Indy系）が複数＋パワー・先行・馬格。芝で切れ負けはダート転換の好材料。
5. **逆ワナ**: 「ダートのクロフネ」だが産駒のダートG1は不在（G1馬ソダシ・カレンチャン等は全て芝）＝産駒は芝マイル＋条件級ダート短距離が本領。イメージで適性を過信しない（as-of 2026-06）。
6. **道悪・成長型は系統で一次フィルタ→着度数データで本判定**（系統名だけの機械的決め打ち禁止。早熟イメージの過剰人気を逆手に取る視点も持つ）。
