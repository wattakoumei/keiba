## 種牡馬別web調査メモ

### マインドユアビスケッツ (1 ユーウェイン父)
- ダート安定型(勝率7.3/連対15.2/複勝20.8)。ダ中距離でスピード持続、ダ1600m以上の外枠で好走率上昇。
- 得意コース筆頭に「東京ダ1600」。母父ディープインパクトはマイル～中距離のスピード/末脚補強でニックス。
- 道悪ダートでパフォ上昇。→ 東京ダ1600適性◎、母父DIで上り強化。総合プラス材料大。
- 出典: thoroughbredskills.com, note浅次郎, 一目でわかる(nobinobi)

### ゴールドドリーム (9 タイセイエムロード父)
- ダート中心の種牡馬。距離は中距離型(1700-2000mがA級)、短距離は酷い成績。マイルも産駒は1700m以上が中心。
- → 東京ダ1600はやや短い側。母父キングカメハメハはパワー/スピード補完でダート向き。地力はあるが距離が微妙。
- 出典: thoroughbredskills.com, note浅次郎, umadb

### Ghaiyyath ガイヤース (10 ザーフィル父)
- 父Dubawi、欧州芝中長距離G1の逃げ強者。産駒のダート適性データは乏しく欠損(確信度↓)。
- 欧州芝中距離血統で、日本ダ1600の適性は未知数。母父Tamayuzも欧州血統。ダート短中距離への直接的プラス材料は薄い。
- 出典: ahonoora.com, Wikipedia, JBIS(具体産駒ダートデータなし)

### ドレフォン (5 ファンタイムギフト父)
- ダ1600mが産駒の距離別最良ゾーン(複勝率40.4%)。「東京ダ1600」がドレフォン産駒の主戦場。
- 母父ダイワメジャーはマイルのスピード/気性強化、ドレフォン×ダイワメジャーは上限の高い配合。距離適性1200-1600(ベスト1400前後)。
- → 本レース条件(東京ダ1600)に最適。最強クラスのプラス材料。
- 出典: navi-keiba, yuruyuru-biyori, keibalist(ダイワメジャー東京ダ1600), umapick

### ミスターメロディ (16 ザローズハーツ父)
- ダート短距離A級(中山ダ1200が鬼)。ダ1000/芝1200で圧倒的、ダート勝ち馬は限定的で距離が延びると割引。
- マイル戦は強い産駒なら一定融通だが基本は短距離。→ 東京ダ1600は距離が長い側、牝馬。母父キングカメハメハはダート対応プラスだが距離克服課題。
- 出典: note浅次郎, thoroughbredskills, nobinobi

### アジアエクスプレス (11 タイセイエキサイト父)
- 短距離寄り(1400以下が明確に得意)。1600m以上は単回30%程度で割引、マイル以上は疑ってかかるべき。
- → 東京ダ1600は適性の外。母父アイルハヴアナザーはダートパワー血統だが距離面のマイナス大きい。
- 出典: nobinobi, thoroughbredskills, note浅次郎

### アメリカンペイトリオット (8 ベンメリア父)
- 父War Front系。実データは芝の方が安定、ダートは人気の割に回収低め。距離は1800m以下、1200-1600で良績。
- ダ中距離(中山/阪神ダ1800、阪神ダ1400、小倉ダ1700)で走るが良馬場向きで道悪注意。
- → ダ1600は守備範囲内だがダート専用ではない、やや軸足は芝。母父ケイムホームはダート短中距離のスピード血統でダート寄りに補完。中立～微プラス。
- 出典: unnantrc, nobinobi, thoroughbredskills

### デクラレーションオブウォー (13 カンレイスキー父)
- 芝ダ兼用、ダート(特に道悪)が狙い目。距離はマイル～2000中距離、ダートは短距離寄りの傾向も。レースセンス・脚質自在。
- → 東京ダ1600は守備範囲。母父ホワイトマズルはスタミナ/欧州血統で渋った馬場で活きる。馬場が渋れば上昇。中立～微プラス。
- 出典: thoroughbredskills, nobinobi, umapick, navi-keiba

### カリフォルニアクローム (6 カルフ父)
- 牡(せん)は明確に砂向き(芝13/ダ26勝、牡はダ15勝)。得意「東京ダ1600」「中山ダ1800」のダ中距離種牡馬。重不良で上昇。
- → 東京ダ1600が得意コードに合致、せん馬で砂寄り。母父スズカマンボ(サンデー系)はスタミナ寄り補完。プラス材料大。
- 出典: nobinobi, 中日スポーツ村本コラム, note浅次郎

### リアルインパクト (14 レッドシュトラール父)
- 芝ダ兼用でむしろダート良績、パワーある産駒。ダートは1700-2000中距離が良いがデータ上「東京ダ1600」で父リアルインパクト産駒が複勝率90.9%級の好データ(人気馬中心)。重馬場も狙い。
- → 東京ダ1600適性は高い部類。母父ジャングルポケットはパワー/底力補完。プラス材料。
- 出典: hikky-keiba, note浅次郎, jra-kenkyu(東京ダ1600データ)

### ルヴァンスレーヴ (3 タキノボリ父)
- ほぼダート専用、パワー型先行。ただし距離は中距離以上(1800中心、2100以上の長距離で上昇)。ダ1800で連対30%前後。
- → 本質ダートだが東京ダ1600は産駒のベスト距離より短い側。母父アグネスデジタルはダート万能のプラス。ダート地力は確かだが距離はやや忙しい。中立～微プラス。
- 出典: nobinobi, thoroughbredskills, taro(note)

### ハービンジャー (12 パッカパッカブー父)
- ダート適性ほぼ無し、ダート戦は基本消し。先行力低く長距離向き、日本ダートと逆方向。例外は阪神ダ2000のみ。
- → 東京ダ1600は明確なマイナス。母父ハーツクライも芝中長距離血統でダート短中距離に逆風。大きなマイナス材料。
- 出典: nobinobi, navi-keiba, dirt_matrix(note ダート苦手種牡馬)

### ミッキーグローリー (4 ドルビナ父)
- 父ディープインパクト、現役は芝マイラー(関屋記念等)。産駒のダート具体データ乏しく欠損(確信度↓)。基本は芝寄り血統。
- → ダート短中距離への直接プラスは薄い。母父カジノドライヴ(ストームキャット系)がダート方向に引っ張る数少ないプラス材料。中立～微マイナス。
- 出典: umadb, jbis, netkeiba(具体ダート傾向は限定)

### アドミラブル (15 オンサイト父)
- 父ディープインパクト。産駒は芝ダ両用で短距離型多い(1000-1400中心)、平均勝利距離ダ1378m。成長型で各場走る。
- → 東京ダ1600は産駒の中心距離(1000-1400)よりやや長い。母父キングズベストは欧州中距離血統。ダート短距離寄りでマイルは微妙。中立。
- 出典: ittai(VictoryRoad), keibalab, kurukuru-keiba

### ポエティックフレア (2 ピースサイン父)
- 父Dawn Approach系、欧州マイルG1馬。産駒は芝マイル中心のS級。ダートは「短距離だけならいける可能性、中距離はさっぱり」「ダートのつぶしが効きにくい」。
- → 東京ダ1600(ダート中距離寄り)はマイナス傾向。母父Le Havre(欧州芝)もダートに逆風。母系次第でダート対応の余地はあるが基本は芝。マイナス材料。
- 出典: umapick, note浅次郎, shadai-ss, gzero

### バトルプラン (7 アイファーファイト父)
- 父Empire Maker系。ダート血統に傾倒、牡は中距離型・牝は短距離型。ダ1800以下中心、ローカル1700も走る。人気薄で妙味あるダート種牡馬。
- → 東京ダ1600は守備範囲(牡の中距離型)。母父ステイゴールドはスタミナ/勝負根性補完。血統的にはダート適性あり。ただし当該馬は7歳・近走大敗続きで血統妙味を現状能力が裏切る(血統観点は中立～微プラスに留める)。
- 出典: navi-keiba, keibalab, Wikipedia, ittai
