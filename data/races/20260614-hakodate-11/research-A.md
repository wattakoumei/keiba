# 観点A 能力指数・時計 調査結果（函館日刊スポーツ杯 / 函館芝1200 洋芝 3勝クラス・16頭）

## 観点の射程
スピード/タイム指数・上がり3F・走破時計・レースレベル。持ちタイムの絶対値だけでなく「どのレベルのレースで出した時計か」を重視。
モデル上は A_class(地力) と A_finish(終盤の決め手) に効く。素の評価のみ返す（相別振り分けは合成側）。

## 確信度・データ制約
- 馬個別の netkeiba/競馬ラボ DB は 403 で直接取得不可。スポーツ紙の公開タイム指数も単独馬では取得できず。
- よって評価の素は **_context.md / 出走表.md の seed（上がり3F・着順・頭数・第1コーナー通過・レース名・日付）= 本ハーネスの近走正本** ＋ web で取れたレースレベルの事実。
- web で確認できた事実: 周防灘特別(2026-02-01 小倉芝1200) は **勝ち時計1:06.9・前半3F 32.5 の真性ハイペース**（keibabook seiseki）。この一戦が複数馬(#1,#9,#14)の物差しになる。
- 鮮度割引: 2年以上前のベストは満額にしない。本レースの近走はほぼ直近半年〜1年内なので鮮度問題は小さいが、好走が古い・1走だけの馬は確信度を下げた。
- レースレベルの定性判断: 3勝クラス戦。2勝クラスで好時計・速い上がりで好走 → 加点。2勝/3勝で上がりかかり負け続け → 減点。1勝クラス勝ち止まりの好走は割引。

## レースレベルの基準メモ
- 周防灘特別(芝1200): 前半32.5の真性ハイ。ここで上位 or 鋭い上がり = 一線級の地力＆決め手の証拠。
- 函館芝1200 洋芝・終盤下り = 前残り・時計かかる。**素の決め手(上がり)より「速いペースを先行して粘れる地力」**が効くコース。上がり最速型は展開待ち（=A_finishは持つがコース相性は合成側判断）。
